# Ejercicios Java POO desde Cero

Este repositorio contiene una recopilación de códigos, ejercicios y prácticas desarrolladas como parte de un curso de programación en Java y Programación Orientada a Objetos.

El contenido está basado en el curso de MoureDev Campus:

https://campus.mouredev.pro/order?ct=6d3d7653-ab53-4f47-a8b5-ffe561a40fdb

Este curso está compuesto por una serie de videos con explicaciones, ejercicios prácticos y actividades de programación. Al completar el curso, se otorga un certificado de finalización.

## Descripción

El objetivo de este repositorio es organizar los ejercicios realizados durante el curso, mostrando el avance en temas fundamentales de Java, estructuras de datos básicas y conceptos principales de Programación Orientada a Objetos.

Los códigos incluidos corresponden a prácticas educativas enfocadas en reforzar la lógica de programación, el uso de Java y la aplicación de conceptos como clases, herencia, polimorfismo, abstracción, composición y manejo de excepciones.

## Contenido del repositorio

En este repositorio se incluyen ejercicios relacionados con:

- Operadores
- Strings
- Condicionales
- Arrays
- Listas
- Sets
- Maps
- Bucles
- Funciones
- Clases
- Modificadores de acceso
- Herencia
- Polimorfismo
- Abstracción
- Composición
- Excepciones
- Ejercicios adicionales y proyectos en IntelliJ IDEA

## Tecnologías utilizadas

- Java
- IntelliJ IDEA
- Programación Orientada a Objetos

## Propósito

Este repositorio tiene fines educativos y personales. Su finalidad es documentar el aprendizaje obtenido durante el curso, servir como evidencia de práctica y funcionar como referencia para repasar conceptos importantes de Java.
